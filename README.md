# 🤖 Telegram AI Bot (Groq & Llama 3)

Мой тренировочный проект: Telegram-бот на базе высокоскоростных моделей Llama от Groq. Проект ориентирован на изучение API-интеграций и основ тестирования (QA).

## 🚀 Функционал
* **Умные ответы**: Использует модель **Llama 3.3 70B** для текстового общения.
* **Стабильность**: Бот настроен на работу с актуальными API-эндпоинтами Groq.
* **Логирование**: В консоль выводятся статусы обработки каждого запроса для удобства отладки.

## 🛠 Технологии
* **Node.js** — среда выполнения.
* **Telegraf** — библиотека для работы с Telegram Bot API.
* **Groq SDK** — движок для работы с нейросетями Llama.
* **dotenv** — безопасное хранение ключей в локальном окружении.

## 📦 Установка и запуск
1. Склонируйте репозиторий.
2. Установите зависимости: `npm install`.
3. Создайте файл `config.env` в корневой папке (этот файл игнорируется Git для безопасности).
