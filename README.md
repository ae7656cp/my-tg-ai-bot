# 🤖 Telegram AI Bot (Gemini 3 Flash)

Мой тренировочный проект: Telegram-бот с глубокой интеграцией искусственного интеллекта от Google.

## 🚀 Функционал
* **Умные ответы:** Отвечает на текстовые сообщения, используя новейшую нейросеть **Gemini 3 Flash Preview**.
* **Зрение (Vision):** Бот умеет анализировать изображения и скриншоты. Просто отправь фото, и он опишет содержимое или поможет найти ошибку в коде.
* **Скорость:** Благодаря модели Flash, ответы приходят практически мгновенно.

## 🛠 Технологии
* **Node.js** — среда выполнения.
* **Telegraf** — современная библиотека для работы с Telegram API.
* **Google Generative AI** — ядро ИИ (версия Gemini 3).
* **dotenv** — для безопасного хранения API-ключей в локальном окружении.

## 📦 Установка и запуск
1. Склонируйте репозиторий.
2. Установите зависимости: `npm install`.
3. Создайте файл `config.env` в корневой папке (этот файл игнорируется Git для безопасности).
4. Добавьте в `config.env` свои ключи:
   ```env
   TELEGRAM_TOKEN=ваш_токен_от_BotFather
   GEMINI_KEY=ваш_ключ_от_Google_AI_Studio
